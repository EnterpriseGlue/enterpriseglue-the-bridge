# Support

## Where to get help

- **Questions / how-to / troubleshooting**: GitHub Discussions
- **Bug reports**: GitHub Issues
- **Feature requests**: GitHub Issues

## Security issues

Do not open public issues for security reports.

See `SECURITY.md` for the preferred reporting process.

## What support covers

Support is provided on a best-effort basis for:

- Reproducible bugs
- Installation and local development questions
- Clarifications about documented behavior

## What support does not cover

- Custom implementation work
- Dedicated 1:1 support or guaranteed response times
- Support for unofficial forks or heavily modified deployments
